# TADS - Unicamp

Este repositório foi criado com o objetivo de centralizar e organizar todos os scripts desenvolvidos durante o curso de Tecnologia em Análise e Desenvolvimento de Sistemas (TADS) na Unicamp.
A ideia é manter um histórico do meu aprendizado em programação ao longo dos semestres.

-----

O projeto está organizado por semestres:

- **1_Semestre**: Introdução à programação, algoritmos e lógica (Scripts em C).
- **2_Semestre**: Avanço na linguagem C, focado em estruturas de dados e manipulação de arquivos (leitura, escrita e persistência de dados).
- **3_Semestre**: Programação Orientada a Objetos (POO), desenvolvimento em Java e tópicos avançados.

-----

Durante as disciplinas, as seguintes ferramentas e linguagens foram exploradas:
- Linguagens de Programação: C, Java
- IDEs/Editores: IntelliJ IDEA, VS Code
- Conexão Remota e Terminal: MobaXterm
- Controle de Versão: Git e GitHub
